# Code of Conduct

BETON follows the [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short:

- Be respectful.
- Disagree on technical substance, not on people.
- No harassment, no personal attacks, no discrimination.

If something happens that shouldn't, report it to szymon.jendryczkos@gmail.com.
